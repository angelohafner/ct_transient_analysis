# CT Saturation Simulator (PSRC) - Streamlit (no sidebar)

Mudancas pedidas:
- Sem barra lateral: todos os controles ficam na aba **Simulacao**
- Espaco para figura: upload no app + opcao de arquivo fixo `assets/figure.png`
- Botao Run simulation mais estreito (nao usa use_container_width)

## Rodar

```bash
pip install -r requirements.txt
streamlit run app.py
```

## Figura

- Opcao 1: use o upload dentro do app
- Opcao 2: coloque sua figura em `assets/figure.png`
